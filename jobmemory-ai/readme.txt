GAIS会員企業向けAI求人文生成プラグイン。

== Description ==

JobMemory AI for GAISは、GAIS（生成AI協会）会員企業向けのAI求人文生成プラグインです。

職種名と自社情報を入力すると、GAISの知見をMemoryとして活用し、3つの異なるトーン（スタンダード・挑戦的・カジュアル）で求人文を自動生成します。

**主な機能:**

* AIによる3パターンの求人文自動生成
* タブUIで3パターンを比較・選択
* Simple Job Boardへの下書き保存
* フィードバックのMemory蓄積（使うほど最適化）
* GAISの知見を初期Memoryとして搭載

**必要なプラグイン:**

* Simple Job Board（求人の保存先として使用）

== Installation ==

1. `jobmemory-ai` フォルダを `/wp-content/plugins/` にアップロード
2. WordPressの「プラグイン」画面から「JobMemory AI for GAIS」を有効化
3. 管理メニューの「JobMemory AI」→「設定」からOpenAI APIキーを入力
4. 「求人生成」画面から求人文を生成
